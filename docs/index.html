como funciona

ML cria o contexto, as LLMs competem, o combo consolida

O agente monta evidencias por partida, chama modelos locais em sequencia, valida cada resposta com Pydantic e salva o payload completo. Respostas fora do JSON esperado ficam auditaveis, mas nao entram no combo.

1. DadosFixtures e classificacao FIFA entram por cache local com refresh sob demanda.
2. PredicaoPoisson, Elo e estatisticas alimentam o prompt estruturado para cada modelo.
3. RankingPalpites travados recebem pontos por placar exato, resultado e Brier score.
